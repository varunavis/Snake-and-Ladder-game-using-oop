This snake and ladder game was developed in Dev-C++ environment using language C++.

Instructions to run the game in your output window
1. Download all 5 of the header files (BasicInfo.h, Board.h, Dice.h, Ladders.h, Snakes.h)
2. Place them in that folder where you've stored your C++ files, or where the compiler and other header files are located
3. Run 'Controller.cpp' (either from downloads, or place them along with the header files and run it)
4. Any other IDE can be used to run the code (eg CodeBlocks, etc)
5. Play the game, and leave a review!

You can use the code for your own purposes but please do give credit!
